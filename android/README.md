# KyroSpades — Android port

SDL2-based Android build of KyroSpades. Native code is the same `src/` tree as
the desktop builds, compiled as `libmain.so` (see `if(ANDROID)` branches in
`src/CMakeLists.txt`). This directory is a normal Android Studio / Gradle
project that points its `externalNativeBuild` at the repo root `CMakeLists.txt`.

## Building locally

```sh
# 1. One-time: cross-compile enet + libdeflate (and verify the native build)
./build_android.sh            # from the repo root; needs the Android NDK

# 2. Assemble game assets into app/src/main/assets/
./package_android_assets.sh   # from the repo root; downloads bsresources.zip if missing

# 3. Build the APK
cd android
./gradlew assembleDebug       # or assembleRelease
```

The KyroSpades source path defaults to `..` (the repo root). To build against a
different checkout, set `kyrospades.path=/path/to/KyroSpades` in
`android/local.properties`.

## CI / releases

The GitHub workflows build the APK alongside the Windows/Linux/macOS zips and
attach it to the release. Release APKs are signed with a real key if these
repository secrets are set, otherwise they fall back to the debug key (still
installable by sideloading):

- `ANDROID_KEYSTORE_BASE64` — `base64 -w0 my-release.keystore`
- `ANDROID_KEYSTORE_PASSWORD`
- `ANDROID_KEY_ALIAS`
- `ANDROID_KEY_PASSWORD`

Generate a keystore once with:

```sh
keytool -genkeypair -v -keystore my-release.keystore -alias kyrospades \
        -keyalg RSA -keysize 2048 -validity 10000
```

Note: an APK signed with a *different* key than the one already installed on a
phone requires uninstalling first. Set the secrets early if you plan to ship
updates.

## Notes

- ABI: `arm64-v8a` only, minSdk 21 (Android 5.0). Add `armeabi-v7a` to
  `abiFilters` in `app/build.gradle` for 32-bit devices.
- Sound is enabled (`-DENABLE_SOUND=ON`) via a static openal-soft 1.23.1 with
  the OpenSL ES backend; wav assets are decoded from APK assets in memory.
- `app/src/main/assets/` is generated by `package_android_assets.sh` and
  gitignored — never commit it.
